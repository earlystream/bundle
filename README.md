# BundleUI

BundleUI is a small client-side Fabric mod for Minecraft `1.21.11` that fixes the vanilla bundle tooltip preview cap.

When you hover a bundle, the tooltip keeps the vanilla slot grid, highlight styling, and progress bar, but it renders every stored stack instead of truncating the view to the default preview.

Supported platform:

- Fabric
- Yarn mappings
- Minecraft `1.21.11`

This mod is client-side only. It does not change bundle storage, item behavior, or server logic.
